# Модели расписания движения трамваев

В проекте реализованы три модели расчёта расписания движения трамваев и графиков работы водителей.  
Все модели формируют выходные данные в виде текстовых файлов и графиков, сохраняемых в соответствующих папках.

**Данные для модели взяты из следующих файлов:**
 - [files/data_for_trains.xlsx](./files/data_for_trains.xlsx) - минимальный интервал, продолжительность маршрута
 - [files/schedule_from_get.xlsx](./files/schedule_from_get.xlsx) - время работы линии и время начала пересменки
 - [files/Timetable.xlsx](./files/Timetable.xlsx) - график работы водителей на месяц с пропусками (для модели timetable_model.py)
---

## Модель 1 — [another_model.py](timetable_and_another/another_model.py)

**Описание:**  
Модель с **постоянным интервалом** между отправлениями (5.4 минуты).

**Расположение:**  
[`another_model.py`](timetable_and_another/another_model.py)

**Входные данные:**
- время работы всей линии  
- время начала пересменки  
- постоянный интервал  
- продолжительность маршрута  
- минимальная и максимальная норма работы  

**Выходные данные:**
- полное расписание движения каждого трамвая — [`output_files/schedule.txt`](./output_files/schedule.txt)  
- график работы каждого водителя в конкретный день  

---

## Модель 2 — [another_model_one/another_model_one.py](./another_model_one/another_model_one.py)

**Описание:**  
Модель с **динамическим интервалом**, который задаётся массивом `intervals_by_time`.  
Реализованы случайные факторы для повышения реалистичности:
- перерывы в парке и на конечной (1–3 минуты);  
- длина маршрута случайна (от 76 до 80 минут).

Модель минимизирует количество недоработок водителей.

**Расположение:**  
[`another_model_one/another_model_one.py`](./another_model_one/another_model_one.py)

**Входные данные:**
- время работы всей линии  
- время начала пересменки  
- переменный интервал  
- продолжительность маршрута  
- минимальная и максимальная норма работы  

**Выходные данные:**
- полное расписание движения каждого трамвая — [`another_model_one/output_files/schedule.txt`](./another_model_one/output_files/schedule.txt)  
- график работы каждого водителя в конкретный день

# График работы водителей

**Описание:**  
Модель предназначена для построения графиков работы водителей на один календарный месяц. 
Пропуски, появившиеся в графике, заполняются автоматически, используя водителей из резерва.

**Расположение:**  
[timetable_and_another/timetable_model.py](./timetable_and_another/timetable_model.py)

**Входные данные:**  
- таблица расписания графиков работы водителей на один календарный месяц

**Выходные данные:**  
- та же самая таблица с заполненными пропусками
